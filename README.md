# 📊 Poll Results Visualizer

## 🔍 Overview
This project analyzes poll/survey data and visualizes results using Python.

## 🚀 Features
- Data Cleaning using Pandas
- Vote Share Calculation
- Bar Chart & Pie Chart Visualization
- Demographic Analysis (Age & Gender)

## 🛠️ Tech Stack
- Python
- Pandas
- Matplotlib
- Seaborn

## 📁 Project Structure
- data/ → Dataset
- outputs/ → Generated charts
- images/ → Screenshots
- main.py → Main script

## ▶️ How to Run
python main.py
