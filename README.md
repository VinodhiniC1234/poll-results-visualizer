# 📊 Poll Results Visualizer

## 🔍 Project Overview
The Poll Results Visualizer is a Python-based data analysis project that converts raw survey or poll data into meaningful insights using visualizations.  
It helps in understanding user preferences, trends, and patterns through simple and clear charts.

---

## 🎯 Objective
- To analyze poll/survey data from CSV files  
- To calculate vote counts and percentage share  
- To visualize results using charts  
- To support data-driven decision making  

---

## 🚀 Key Features
- Data loading and preprocessing using Pandas  
- Vote count and percentage calculation  
- Bar chart visualization  
- Pie chart visualization  
- Age-wise analysis  
- Gender-wise analysis  
- Automatic chart saving in outputs folder  

---

## 🛠️ Tech Stack
- Python  
- Pandas  
- Matplotlib  
- Seaborn  

---

## ⚙️ How It Works
1. Load dataset from CSV file  
2. Clean and preprocess the data  
3. Calculate vote counts and percentages  
4. Perform demographic analysis (age & gender)  
5. Generate visualizations (bar chart, pie chart, etc.)  
6. Save outputs as images
